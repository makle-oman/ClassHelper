# ClassHelper - 教师助手 APP 需求文档

## 项目概述

面向小学教师的教学管理工具，帮助老师管理班级、学生、课程、成绩、考勤等日常工作，后续扩展家长端实现家校互动。

## 技术栈

- **前端**：React Native + Expo（TypeScript）
- **后端**：NestJS（TypeScript）
- **数据库**：MySQL + TypeORM
- **部署**：自有服务器 + Nginx + PM2

## 第一阶段 - 教师端 MVP

### 1. 用户模块

- [ ] 教师注册（手机号 + 密码）
- [ ] 教师登录（JWT Token 认证）
- [ ] 个人信息管理（姓名、头像、学校、科目）
- [ ] 修改密码
- [ ] 退出登录

### 2. 班级管理

- [ ] 创建班级（年级 + 班号，如：三年级2班）
- [ ] 编辑/删除班级
- [ ] 查看我管理的班级列表
- [ ] 一个教师可管理多个班级

### 3. 学生管理

- [ ] **Excel 批量导入学生**（核心功能）
  - 提供 Excel 模板下载
  - 模板字段：学号、姓名、性别、出生日期、家长姓名、家长手机号、家长关系（父亲/母亲/其他）
  - 导入时自动校验数据格式
  - 支持导入错误提示（如手机号格式不对、学号重复等）
  - 导入后自动创建学生记录和关联家长信息
- [ ] 手动添加单个学生（作为补充）
- [ ] 编辑学生信息
- [ ] 删除学生（软删除）
- [ ] 按班级查看花名册
- [ ] 学生详情页（基本信息 + 家长信息 + 成绩 + 考勤汇总）

### 4. 学期管理

数据层级关系：
```
学校
 └── 学年（如 2025-2026）
      └── 学期（第一学期 / 第二学期）
           ├── 课程表（绑定班级 + 老师）
           ├── 考试 & 成绩
           └── 考勤记录
```

- [ ] 创建学期（名称、起止日期、共几周、所属学年）
- [ ] 设置当前活跃学期
- [ ] 切换查看历史学期（课程表页顶部可切换学期，默认显示当前学期）
- [ ] 学期归档（归档后数据只读，成绩/考勤/课程表等保留可查）
- [ ] 学期列表（活跃 / 已归档）
- [ ] 课程表绑定学期（每个学期有独立课程表，切换学期可查看历史课程表）
- [ ] 操作入口：「我的」→ 班级管理 → 学期管理

### 5. 课程表管理

- [ ] 课程表绑定学期（每学期独立课程表）
- [ ] 配置每日课程节数和时间段（含早读、午阅、课后服务等特殊时段）
- [ ] 按周设置课程安排（周一至周五）
- [ ] 标记我的课程（当前老师负责的课）
- [ ] 支持调课（临时修改某天某节课）
- [ ] 假期标记（节假日整列标灰）
- [ ] 今日课程视图 + 周课程表视图
- [ ] 课前提醒通知（可配置开关 + 提前时间）

### 6. 年级升迁与毕业归档

升迁流程：
```
班级（三年级2班）→ 年级升迁 → 班级（四年级2班），学生跟随
六年级结束 → 毕业归档 → 数据保留，从活跃列表移除
```

- [ ] 年级升迁（如三年级2班 → 四年级2班），学生跟随升迁
- [ ] 支持一键批量升级（所有班级同时升一个年级）
- [ ] 毕业归档（六年级结束后整届归档）
- [ ] 归档数据保留，仅从活跃列表移除，可随时查看
- [ ] 查看历史归档班级和学生数据（只读模式）
- [ ] 操作入口：「我的」→ 班级管理 → 年级升迁

### 7. 考勤管理

- [ ] 按日期、班级进行考勤
- [ ] 考勤状态：出勤 / 迟到 / 早退 / 请假 / 缺席
- [ ] 批量考勤（全班默认出勤，只标记异常）
- [ ] 查看某天考勤记录
- [ ] 考勤统计
  - 按学生：月度/学期出勤率
  - 按班级：每日出勤率

### 8. 成绩管理

- [ ] 创建考试（考试名称、科目、日期、满分）
- [ ] 录入成绩（支持批量录入）
- [ ] 支持 Excel 导入成绩
- [ ] 自动统计
  - 班级平均分、最高分、最低分
  - 分数段分布（90-100、80-89、70-79、60-69、60以下）
  - 及格率、优秀率
- [ ] 成绩趋势图（多次考试对比）
- [ ] 单个学生成绩详情

### 9. 作业管理

- [ ] 发布作业（班级、科目、内容描述、截止日期）
- [ ] 查看已发布的作业列表
- [ ] 记录作业提交情况（已交 / 未交 / 迟交）
- [ ] 作业完成率统计

### 10. 通知公告

- [ ] 发布通知（标题、内容、通知类型、目标班级）
- [ ] 通知类型：普通通知 / 放假通知 / 活动通知 / 紧急通知
- [ ] 查看已发布的通知列表
- [ ] 通知状态追踪（已读/未读统计）

## 第二阶段 - 家长端

### 9. 家长用户模块

- [ ] 家长注册/登录（手机号，需与导入的手机号匹配）
- [ ] 绑定孩子（通过手机号自动匹配）
- [ ] 一个家长可关联多个孩子

### 10. 家长查看功能

- [ ] 查看孩子的成绩记录和趋势
- [ ] 查看孩子的考勤记录
- [ ] 接收老师发布的通知
- [ ] 查看作业内容

### 11. 家校互动

- [ ] 在线请假申请（家长提交，老师审批）
- [ ] 通知确认回执

## 数据库表设计

| 表名 | 说明 | 主要字段 |
|------|------|----------|
| teachers | 教师 | id, phone, password, name, avatar, school, subject |
| classes | 班级 | id, grade, class_number, teacher_id, is_archived, promoted_from_id |
| students | 学生 | id, student_no, name, gender, birth_date, class_id |
| parents | 家长 | id, phone, name, relationship, student_id |
| courses | 课程表 | id, class_id, weekday, period, subject, teacher_id, semester_id |
| semesters | 学期 | id, name, start_date, end_date, weeks_count, academic_year, is_active, is_archived, teacher_id |
| attendance | 考勤 | id, student_id, date, status, remark |
| exams | 考试 | id, name, subject, date, full_score, class_id |
| scores | 成绩 | id, exam_id, student_id, score |
| homework | 作业 | id, class_id, subject, content, deadline |
| homework_records | 作业记录 | id, homework_id, student_id, status |
| notices | 通知 | id, title, content, type, class_id, teacher_id |
| notice_reads | 通知阅读 | id, notice_id, parent_id, read_at |
| leave_requests | 请假 | id, student_id, parent_id, start_date, end_date, reason, status |
